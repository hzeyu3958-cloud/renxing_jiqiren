# 人形机器人初步

这是一个基于 Webots 的人形机器人仿真项目。

## 项目结构

- `controllers/` - 机器人控制器代码
  - `walk_controller/` - 行走控制器
- `meshes/` - 机器人3D模型网格文件
- `protos/` - Webots PROTO 定义文件
- `worlds/` - Webots 仿真世界文件

## 使用方法

1. 安装 Webots 仿真软件
2. 打开 `worlds/mecha_humanoid_21dof.wbt` 世界文件
3. 运行仿真查看机器人行走

## 机器人规格

- 21自由度人形机器人
- 包含完整的身体部件：头部、躯干、四肢等
